# Ghost Mentor — Agent System

Lyzr-powered multi-agent orchestration layer.

## Agents
| Agent | Responsibility |
|---|---|
| Memory Agent | Stores and retrieves memories |
| Reflection Agent | Detects emotional patterns |
| Career Coach Agent | Provides career guidance |
| Productivity Agent | Tracks consistency and goals |
| Motivation Agent | Encourages and supports users |
| Retrieval Agent | Performs semantic memory search |

## Stack
- Python 3.11+
- Lyzr SDK
